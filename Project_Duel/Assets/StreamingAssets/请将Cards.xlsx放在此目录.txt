请将卡牌表格 Cards.xlsx 放在本目录（StreamingAssets）下。

表格列（第一行为表头）：
id、角色名称、势力、花色、点数、
技能名称一、技能描述一、技能名称二、技能描述二、技能名称三、技能描述三、
是否有技能tag、技能tag

- 技能名称/描述可为空。
- 是否有技能tag：填 true/是/1 或 false/否/0。
- 技能tag：多个用英文竖线 | 分隔。

图鉴展示上限 = 表格中最大的 id 值（只展示 id 从 1 到该最大 id 的卡牌）。
卡面图片请放在 Resources/Cards/ 下，命名为 NO001、NO002…（与 id 对应）。

【技能 tag 悬停介绍】
将 intro.xlsx 放在本目录下，用于图鉴详情里技能描述中 tag 的悬停说明。
- Cards.xlsx 中每个技能的 tag 列（普通形态 H/K/N 列，特殊形态 T/W/Z 列）用 | 分隔多个 tag，如「防御技|主动技」。
- 当鼠标悬停在技能描述里某段与 tag 完全对应的文字上（如「防御技」或「主动技」）0.5 秒后，会弹出该 tag 的说明。
- intro.xlsx 第一行为表头，列名需包含：id（与 tag 完全一致，如 防御技、主动技）、介绍内容 或 内容。每个 tag 一条介绍，悬停到哪个字就显示哪个 id 对应的介绍。
