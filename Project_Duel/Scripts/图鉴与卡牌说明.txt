图鉴与卡牌使用说明（表格驱动）
================================

一、卡牌数据：.xlsx 表格
1. 将表格命名为 Cards.xlsx，放在 Assets/StreamingAssets/ 目录下。
2. 第一行为表头，列名必须包含（顺序可打乱）：
   id、角色名称、势力、花色、点数、
   技能名称一、技能描述一、技能名称二、技能描述二、技能名称三、技能描述三、
   是否有技能tag、技能tag
3. 技能名称/描述可为空。
4. 是否有技能tag：填 true/是/1 或 false/否/0。
5. 技能tag：多个用英文竖线 | 分隔。
6. 图鉴展示上限 = 表格中最大的 id。只会展示 id 从 1 到该最大 id 的卡牌（NO001～NO{maxId}）。

二、卡面图片
- 放在 Resources/Cards/ 下，命名为 NO001、NO002、NO003…（与表格中的 id 对应）。
- 图片格式 .png/.jpg 等由 Unity 导入即可。

三、筛选
- 图鉴页面右上角「筛选」按钮打开筛选面板。
- 左上角为「输入名称」文本框；下方四行分别为势力、花色、点数、所属扩展包的勾选框（多选）。
- 所属扩展包选项：基础、智包、勇包、姿包、阵包、势包、命包、梦包、sp1三足鼎立。
- 多条件为 AND：仅显示同时满足所有已选条件的卡牌。点击「确定」应用筛选并刷新图鉴。

四、卡牌详情
- 点击图鉴中的卡牌，弹出遮罩并在屏幕中央放大显示该卡；点击任意处关闭。
